staticSH.sendPush(params, callback)

/Users/XujieSong/Documents/SK8Tech/Dropbox/#SK8Tech/Tech/Web/ShelfSDK/docs/sources/js/shelf.js, line 347

Send an Push Notification to the given receiver

Name Type Description
params Object

The data of the push notification. Valid fields are:
channels - An Array of channels to push to.
push_time - A Date object for when to send the push.
expiration_time - A Date object for when to expire the push.
expiration_interval - The seconds from now to expire the push.
where - A AV.Query over AV.Installation that is used to match a set of installations to push to.
cql - A CQL statement over AV.Installation that is used to match a set of installations to push to.
data - The data to send as part of the push
callback Object optional

An object that has an optional success function, that takes no arguments and will be called on a successful push, and an error function that takes a AV.Error and will be called if the push failed.
See:
  • https://leancloud.cn/docs/js_push.html#Demo_及示例代码
